The SMART accronym is an outline which you can follow to set strong goals for yourself. Click the letters below to learn about the SMART accronym.


S

Specific:

Goal should be well defined, clear, and unambiguous
M

Measurable:

Goal should have specific measurable criteria demonstrating progress
A

Attainable:

Goal should not be impossible to achieve
R

Relevant:

Goal should be relevant to what you are aiming to improve or accomplish
T

Time-Bound:

Goal should have a clearly defined time line
